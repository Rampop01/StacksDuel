# StacksDuel

**StacksDuel** is a high-interaction, decentralized battleground built on the Stacks blockchain. It leverages Clarity smart contracts to create a low-friction, high-engagement environment for binary voting and on-chain status rewards.

## Vision
To create a "reasonable" but addictive social utility on Stacks. Unlike traditional voting platforms, **StacksDuel** integrates a robust NFT reward system:
- Participant Badges: Earned by every user who votes on a duel.
- Champion Badges: Earned by creators whose predictions are proven correct.

## Tech Stack
- Smart Contracts: Clarity (SIP-009 Compatible)
- Frontend: Next.js 16, TypeScript, Stacks.js
- Architecture: Separated Logic (Traits, NFT implementation, and Duel Engine)

## Repository Structure
- /backend/contracts: Professional Clarity smart contracts for the Stacks Mainnet.
- /frontend: Responsive Next.js dashboard for live battle monitoring and wallet interaction.

## Leaderboard Optimization
This project is optimized for the **Stacks Builder Rewards**, focusing on:
1. Transaction Volume: Every action triggers a dual-transaction (Vote + NFT Mint).
2. Unique Callers: Low friction design attracts a wide array of users.
3. Mainnet Fees: Sophisticated contract logic ensures meaningful gas generation.
